Hobos!

A simple utility that gives you a random hobo name.

git clone https://www.github.com/rhgraysonii/hobos.git
cd hobos
gem build hobos.gemspec
gem install hobos-0.0.1.gem
cd ..

And now, random hobos, or find hobo by hobo id.

hobos
#=> random_hobo_name
hobos 1
#=> Stewbuilder Dennis
